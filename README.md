# Quantum Reinforcement Learning for Lattice-Based Post-Quantum Cryptography

## Description
Quantum Reinforcement Learning (QRL) optimizes lattice-based post-quantum cryptography parameters using Ramanujan noise. Balances security, runtime, memory, and energy efficiency.

## Features
- Adaptive lattice parameter optimization via QRL
- Ramanujan-theoretic noise modeling for LWE schemes
- Supports Kyber and FrodoKEM
- Hardware-aware energy proxy evaluation
- Security guarantees comparable or superior to NIST defaults

## Requirements
- Python 3.9+
- NumPy, SciPy, Matplotlib
- Optional: Qiskit or PennyLane for quantum policy simulation
- -Oqs
